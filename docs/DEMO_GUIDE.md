## 시연 시나리오 (영상 촬영용)

### 시나리오 1: 수강생 — AI 챗봇 대화

1. `student_001` / `student_001`로 로그인
2. 좌측 메뉴에서 **"AI 챗봇"** 클릭
3. 아래 질문들을 순서대로 입력하여 시연:
   - `"오늘 큐레이션 뭐 있어?"` → 오늘의 큐레이션 카드 응답 확인
   - `"채용 공고 알려줘"` → RAG 기반 지식 검색 응답 확인
   - `"보충수업 예약하고 싶어"` → 예약/취소 선택 메뉴 표시
     - **"예약하기"** 클릭 → 날짜 선택 → 시간 선택 → 필요 내용 입력 → 예약 확정
   - `"멘토 상담 요청"` → Human Handoff (1:1 멘토 연결 요청)

### 시나리오 2: 수강생 — 학습 자료 조회

1. 챗봇에서 `"최신 학습 자료 보여줘"` 입력
2. 멘토가 업로드한 최신 자료 카드 표시 확인
3. **"원문 보기"** 또는 **"다운로드"** 버튼 클릭하여 자료 접근

### 시나리오 3: 수강생 — 예약 취소

1. 챗봇에서 `"예약 취소하고 싶어"` 입력
2. 예약/취소 선택 메뉴에서 **"취소하기"** 클릭
3. 기존 예약 목록 표시 → 취소할 예약 선택 → 취소 완료

### 시나리오 4: 멘토 — 대시보드 & 자료 관리

1. `mentor_001` / `mentor_001`로 로그인
2. 대시보드에서 확인:
   - **수강생 활동 타임라인**: 학생별 최근 질문·예약·상담 이력
   - **1:1 상담 요청 알림**: 수강생이 요청한 멘토 상담 대기열
3. **"자료 업로드"** 섹션에서:
   - **최신 자료 업로드**: PDF 파일 선택 → AI가 자동으로 제목/요약 생성
   - **기초 자료 업로드**: 기초 학습 자료 등록
4. 업로드된 자료 옆 **휴지통 아이콘**으로 삭제 시연 (벡터스토어 자동 동기화)

### 시나리오 5: 조교 — 스케줄 관리 & 브리핑

1. `ta_jung` / `ta_jung`으로 로그인
2. **스케줄 관리** 탭에서:
   - AI 챗봇으로 자연어 입력: `"4월 14일부터 18일까지 매일 14시~17시 1시간 단위로 열어줘"`
   - 달력에 스케줄 자동 생성 확인
3. **예약 현황** 탭에서:
   - 학생 예약 목록 확인
   - AI 브리핑 리포트 클릭 → 학생 학습 분석 내용 확인

### 시나리오 6: 관리자 — 큐레이션 관리

1. `admin_001` / `admin_001`로 로그인
2. **큐레이션 관리** 화면에서:
   - **새 큐레이션 등록**: 카테고리 선택(IT뉴스/채용정보/자격증/개발트렌드) → 내용 입력 또는 파일 업로드
   - AI가 자동으로 제목·요약 생성 확인
   - 수정/삭제 시연
3. **멘토 대기열** 확인: 수강생의 1:1 상담 요청 현황

1. 카카오톡에서 **Edu-Sync AI** 채널 검색 → 채팅
2. 웹 챗봇과 동일한 AI 응답 확인
3. 카카오톡 ↔ 웹 어디서든 동일한 서비스 이용 가능

## 핵심 기술 포인트 (영상 설명용)

| 기술                   | 설명                                               |
| ---------------------- | -------------------------------------------------- |
| **멀티 에이전트**      | Router → Agent A(행정) / Agent B(조교) / Handoff(멘토) 자동 분기 |
| **RAG 검색**           | FAISS 벡터스토어로 의미 기반 문서 검색             |
| **AI 브리핑 리포트**   | 학생 학습 이력 분석 → 조교용 사전 브리핑 자동 생성 |
| **AI 자동 요약**       | 업로드된 PDF/파일을 AI가 제목·요약 자동 생성       |
| **4역할 대시보드**     | 수강생·멘토·조교·관리자 각각 전용 UI 제공          |
| **카카오톡 통합**      | 별도 앱 없이 카카오톡에서 동일한 AI 챗봇 이용      |
