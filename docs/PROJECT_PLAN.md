# Edu-Sync AI 프로젝트 기획서

> KDT(K-Digital Training) 교육 운영을 위한 멀티 에이전트 AI 학습 지원 시스템

---

## 1. 프로젝트 개요

**Edu-Sync AI**는 국비지원 코딩 학원(KDT) 운영에 필요한 학습 지원·행정 안내·조교 예약·멘토 상담을 하나의 AI 챗봇 인터페이스로 통합한 플랫폼입니다.

관리자·멘토·조교·수강생 총 4개 역할에 맞춘 전용 대시보드를 제공하며, 카카오 로그인과 웹 로그인을 모두 지원합니다.

## 3. 솔루션 — 핵심 기능

### 3-1. 멀티 에이전트 AI 챗봇

학생이 메시지를 보내면 **Router AI**가 의도를 분류하여 적절한 에이전트로 라우팅합니다.

```
학생 메시지
    ↓
[Router AI] 의도 분류
    ├── 행정/커리어/큐레이션 → [Agent A] RAG 기반 지식 검색 + 답변
    ├── 학습 질문/보충수업 예약 → [Agent B] 스케줄 관리 + 브리핑 생성
    └── 사람 상담 필요 → [Human Handoff] 멘토에게 1:1 알림
```

**Agent A (행정·학습 자료 에이전트)**
- RAG(검색증강생성)로 학원 공지, 채용정보, 뉴스 검색
- 멘토가 올린 학습 자료(최신/기초) 검색 및 원문 제공
- 큐레이션 컨텐츠 자동 안내

**Agent B (조교 스케줄러 에이전트)**
- 4단계 예약 플로우: 날짜 선택 → 시간 선택 → 필요 내용 입력 → 예약 확정
- 예약/취소 선택 메뉴 제공
- 예약 완료 시 AI 브리핑 리포트 자동 생성

**Human Handoff (멘토 연결)**
- AI가 감정 상담·진로 고민 등 사람 상담이 필요한 상황 감지
- 담당 멘토에게 즉시 알림 → 1:1 상담 연결

### 3-2. AI 큐레이션 자동 정리

- 관리자가 PDF/링크를 올리면 AI가 **제목·요약을 자동 생성**
- 카테고리별(IT뉴스/채용정보/자격증/개발트렌드) 자동 분류
- 벡터스토어에 저장되어 학생 질문 시 자동 검색·제공

### 3-3. 조교 스케줄 관리 + AI 브리핑

- 자연어로 스케줄 생성: `"4월 14일부터 18일까지 14시~17시 1시간 단위"` → 자동 생성
- 달력 UI에서 예약 현황 시각화
- **AI 브리핑 리포트**: 학생의 과거 질문·학습 이력을 분석하여 조교에게 사전 전달

### 3-4. 멘토 자료 업로드 + 챗봇 자동 연동

- PDF/파일 업로드 시 AI가 제목·요약 자동 생성
- **최신 자료 / 기초 자료** 분리 관리
- 업로드 즉시 벡터 데이터베이스에 반영 → 학생 질문에 자동 검색

### 3-5. 카카오톡 채널 연동

- 카카오톡 채널로 접속해도 웹 챗봇과 **동일한 AI 에이전트가 응답**
- 별도 앱 설치 없이 카카오톡에서 즉시 이용 가능
- 웹 ↔ 카카오톡 어디서든 동일한 서비스 경험

## 4. 역할별 대시보드

| 역할 | 주요 기능 |
| ---- | --------- |
| **수강생** | AI 챗봇, 오늘의 큐레이션, 학습 팁(최신/기초), 조교 예약·취소, 1:1 멘토 상담 요청 |
| **멘토** | 수강생 활동 타임라인, 조교 연결 현황, 자료 업로드(최신/기초), 큐레이션 확인, 수강생 초대 링크, 1:1 상담 요청 알림 |
| **조교** | 월간 달력 스케줄 관리(AI챗봇/수동), 예약 현황, AI 브리핑 리포트 확인 |
| **관리자** | 큐레이션 CRUD(등록/수정/삭제), 멘토 대기열 확인 |

## 5. 기대 효과

### 수강생
- 24시간 AI 즉시 응답으로 대기 시간 제거
- 채용·뉴스·공모전 정보가 챗봇에서 자동 노출 → 정보 누락 감소
- 보충수업 예약이 3스텝 이내 완료
- 카카오톡에서 바로 이용 → 접근성 극대화

### 멘토
- 학생별 질문 기록·예약·활동을 대시보드 한 화면에서 추적
- 데이터 기반 학생 파악 → 면담 준비 시간 절감
- AI가 감지 못하는 감정·진로 상담에 선제적 개입 가능

### 조교
- 자연어 스케줄 설정 → 관리 시간 대폭 감소
- AI 브리핑으로 "학생이 뭘 모르는지" 수업 전 사전 파악

### 관리자
- 파일/링크만 올리면 AI가 정리·배포 → 큐레이션 업무 자동화
- 학생 참여 데이터가 자동 축적

## 6. 향후 확장 계획

- 학습 분석 대시보드: 학생별 학습 패턴 시각화, 위험 학생 조기 감지
- 그룹 스터디 매칭: AI 기반 유사 수준 학생 그룹 자동 구성
- 포트폴리오 자동 생성: 학습 이력 기반 취업용 포트폴리오 초안 생성
- On-premise LLM: 보안이 중요한 교육기관용 자체 서버 AI 모델 전환
